# MetroCLI

Natural language terminal TUI built with Node.js, TypeScript, React, and Ink.

## Run

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
pnpm start
```

## Environment

```text
NVIDIA_API_KEY=your_key_here
NVIDIA_BASE_URL=https://integrate.api.nvidia.com/v1
NVIDIA_MODEL=z-ai/glm-5.2
```

## Useful Commands

```text
/help
/context
/files
/history
/model
/reset
```
